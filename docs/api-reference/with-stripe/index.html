withStripe

The withStripe function is a higher-order component that injects a stripe prop into the component given to it as an argument. The stripe prop is an instance of the Stripe object, and should be used in components dealing with payment flows.

Import#

import { withStripe } from '@staqjs/client'

Example Usage#

import { withStripe } from '@staqjs/client'

function Dashboard(props) {
  const { stripe } = props

  // ... code ...
}

export default withStripe(Dashboard)
